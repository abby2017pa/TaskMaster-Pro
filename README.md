# TaskMaster Pro

Welcome to **TaskMaster Pro** – your ultimate solution to organize, prioritize, and manage your daily tasks efficiently.

---

## Overview

TaskMaster Pro is designed to help you stay on top of your work and personal tasks with a clean, intuitive interface and powerful features that keep you productive throughout the day. Built with modern web technologies, it offers seamless task creation, categorization, and reminders.

## Features

- Create, edit, and delete tasks effortlessly
- Organize tasks into customizable categories
- Set deadlines and receive timely reminders
- Prioritize tasks with color-coded importance
- Responsive design for desktop and mobile

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/abby2017pa/taskmaster-pro.git
   ```
2. Navigate to the project directory:
   ```bash
   cd taskmaster-pro
   ```
3. Install dependencies for frontend and backend:
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```
4. Set up environment variables in `backend/.env` (e.g., DB URI, JWT secret).
5. Start backend server:
   ```bash
   npm start
   ```
6. Start frontend server:
   ```bash
   npm start
   ```

## Usage

- Navigate to `http://localhost:3000` in your browser.
- Register a new account or log in with existing credentials.
- Add new tasks by clicking the "Add Task" button.
- Assign tasks to categories and set deadlines.
- Mark tasks as complete to keep track of progress.
